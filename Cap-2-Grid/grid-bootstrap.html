Grid-Bootstrap

O Grid Bootstrap é construído com CSS FlexBox. onde cada linha é um container flex e cada coluna, um flex item. A distribuição das colunas dentro da linha por padrão, é na horizontal flex-direction:row. Assim, na teoria podemos adicionar infinitas colunas em uma linha.

Todo grid deve ser marcado dentro de um elemento-pai ao qual se atribui a classe container ou container-fluid. A primeira cria um grid, cuja largura é determinada pelos breakpoins, centralizado na viewport, enquanto a segunda cria um grid cuja largura é igual a largura da viewport. Existem ainda classes cuja sitaxe geral é -{xs|sm|md|lg|xl|xxl} que serão apresentadas mas a frente.

O elemento que recebe a class="row" é o contêiner de uma linha do grid e, por padrão um flex contêiner.

Já o elemento que recebe a class="col" é uma coluna, que, por estar contida em um flex contêiner, é um flex item

As siglas em inglês nomeiam as faixas de largura em cada breakpoint, conforme é descrito a seguir

  • xs <= 576px - largura extra pequena
  • sm >= 576px - largura pequena
  • md >= 768px - largura média
  • lg >= 992px - largura grande
  • xl >= 1200px - largura extra grande
  • xxl >=1400x - largura extra extra grande

Classes para colunas

No exemplo anterior definimos uma classe com o valor col para cada contêiner de cada coluna, que se pose inserir em uma linha div.row Entretanto, existe a possibilidade de inserir em uma linha div.row colunas com larguras iguais ou diferentes usando a classe cuja sintaxe geral é col-n onde n é um número inteiro maior que 1 e menor ou igual a 12.

O número após o prefixo col- define a quantidade de colunas a mesclar para obter uma célula de conteúdo. Podemos notar que dentro da linha div.row a soma dos números constantes do prefixo é igual a 12 4 + 1 + 5 + 2. Se esse número for ultrapassado o máximo de 12 colunas havera uma quebra de linha

Prefixo de classes com breakpoint

As classes para colunas possibilitam que se defina a mesclagem de colunas com base nas faixas de largura da viewport(breakpoins), tornando o grid responsivo e adaptável às diferentes larguras de viewport.

Gutters

São espaços horizontais entre as colunas em um layout de grid. Eles desempenham um papel crucial no espaçamento e alinhamento adequados dos elementos do layout.

São utilizados para criar uma separação visual entra as colunas nas adjacentes, evitando que os elementos se sobreponham uns aos outros. ELes fornecem um spaço consistente entre as colunas, garantindo um design coeso e orfanizado.

A aplicação do Gutter deverá ser feita no contêiner da linha .row. Se precisar é possivel definir gutter na vertical

A sintaxe utilizada para definição de Gutters é :

.gx{número} e .gy-{número} para gutter horizontal e vertical respectivamente, sendo o placeholder{número} um número de 0 a 5 que define Gutters de 0.25rem, 0.5rem, 1rem, 1.5rem e 3rem.
