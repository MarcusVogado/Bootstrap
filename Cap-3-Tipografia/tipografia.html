TIPOGRAFIA

Introdução

Vamos estudar a estilização padrão do Bootstrap para classes tipográficas como .h1, .h2, .p, .lead, para classes de estilização de texto como negrito, itálico, sublinhado e tachado, também para classes de alinhamento de texto e espaçamento entre linhas e letras

Família de fontes

Por padrão o bootstrap define para o element body um conjunto de fontes conhecido como native font stack ou system font stack. Trata-se de uma lista que já vem configurada com valores para a propriedade font-family, nesta configuração comtempla pelomenos uma fonte compatível com o sistema operacional que o usuário está utilizando. Esta configuração pode ser encontrada no arquivo bootstrap.css nas primeiras linhas do código, ela é definida como uma variável CSS pelo seletor :root{--bs-font-sans-serif:}

Podemos importar novas vontes do Google Fonts por exemplo e fazer a configuração normalmente criando uma folha de estilo customizada com regra CSS que sobrescreva a regra padrão.

Estilização dos elementos h1 até o h6 e p

Os tamanhos podroes das fontes são:

  • h-1, .h-1{font-size: 2.5rem}
  • h-2, .h-2{font-size: 2rem}
  • h-3, .h-3{font-size: 1.75rem}
  • h-4, .h-4{font-size: 1.5rem}
  • h-5, .h-5{font-size: 1.25rem}
  • h-6, .h-6{font-size: 1.rem}

Por padrão a propriedade font-weight para esses elementos tem o valor 500(que seria um semi-bold) e para line-height tem o valor 1.2

Ao utilizar as classes que o bootstrap disponibiliza para elementos h1 até o h6, podemos utilizar essas classes em qualquer elemento e ela tera o mesmo comportamento de um h1 por exemplo: <span class="h1" /> Texto do elemento span </span>

Classes para textos

Precisamos resaltar que ao utilizar essas classes de forma excessiva pode resultar em um código extremamente poluído e difícil de manter. É recomendado utilizar essas classes com moderação para sempre buscar o equilíbrio.

Alinhamento

As classes .text-start, .text-center e .text-end servem para alinha os elementos dentro do seu contêiner. Podemos atrelar o alinhamento dos texto à largura da viewport usando uma sintaxe modifica para essas propriedades, que consiste em acrescentar a sigla do breakpoint. por exemplo: .text-md-center, .text-lg-end etc.
